Detailed Description

Responsible for the LD matrix.

This class is responsible for anything related to the LD matrix. Most importantly, it is responsible for the construction of LD matrix and the core decomposition step of the matrix equation. The most complicated part of this class is the part to deal with perfect LD. We take additional windows so that we make sure that for any window that were processing, their perfect LD partners are taken into account. This is based on the assumption that there will be no true perfect LD between two Snps if they are 1 window size away from each others.
