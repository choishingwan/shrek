Detailed Description

The region class, store the variance information of each region.

The main goal of this class is to allow user to provide multiple bed files when they would like to run the programme so that heritability estimation can be performed on all the regions at once. e.g. different snp categories, different gene sets etc. This class will also form the basis to construct the flag information of the Snps. The first region is the default region, which contains all the snps that have the LD information.
